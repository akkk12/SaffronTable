# Saffron Table

Modern refresh of the original 2020 restaurant webpage.

## What changed

- Rebuilt the page with semantic HTML, modern CSS, and ES modules.
- Removed jQuery and Bootstrap in favor of lightweight native browser APIs.
- Added responsive navigation, accessible focus states, skip link, and ARIA status messaging.
- Converted the menu into data-driven rendering with category filters.
- Upgraded the reservation form with native date/time inputs, validation, Monday closure logic, and no-refresh feedback.
- Added a stronger visual system, first-viewport restaurant photography, responsive grids, and cleaner content hierarchy.

## Why it fits a 3-year SDE portfolio

This version shows product thinking and engineering maturity beyond a static landing page: component-like data rendering, accessibility, progressive enhancement, maintainable CSS tokens, form state handling, and no unnecessary framework dependency for a small surface area.

## Run locally

Open `index.html` in a browser. No build step is required.
